# No External Fragmentation!

## Problem statement
In a file allocation system, which of the following allocation scheme(s) can be used if no external fragmentation is allowed?
```
I. Contiguous
II. Linked
III. Indexed
```

### Options: Pick one correct answer from below

  - I and III only
  
  - II only

  - III only

  - II and III only

---

## Solution Description
*4* ==> The contiguous disc allocation cannot be used because there is external fragmentation due to residual memory left after the allocation of the process in the memory. The linked list and index disc allocation use pointers to store the location of the next memory block so it will be free from external fragmentation.
