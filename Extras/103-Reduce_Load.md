# Reduce Load 

## Problem statement
In system design, the proxy server reduces the load on _______.

## Options: Pick one correct answer from below

  - Proxy server

  - Node server

  - Domain server

  - Original server

---

# Solution description
*4* --> In a system design, the proxy server reduces the load on the original server. By handling requests and caching responses, the proxy server can reduce the number of direct requests that reach the original server, improving overall performance and efficiency.
