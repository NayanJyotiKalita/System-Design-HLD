# Reverse proxy

## Problem statement
What is the main purpose of using a reverse proxy in a system design?

## Options: Pick one correct answer from below

  - To reduce server load

  - To handle client requests directly

  - To improve security by hiding the backend servers

  - To increase database performance

---

# Solution description
*3* --> A reverse proxy forwards client requests to backend servers, providing an additional layer of security and load distribution.
