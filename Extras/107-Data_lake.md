# Data lake

## Problem statement
In system design, what is the purpose of using a data lake?

## Options: Pick one correct answer from below

  - To store structured data in relational databases

  - To manage transactional data

  - To store large volumes of raw, unstructured data

  - To optimize real-time data processing

---

# Solution description
*3* --> A data lake stores raw, unstructured data from various sources, allowing for flexible data processing and analysis.
