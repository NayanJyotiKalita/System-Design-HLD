# Eventual consistency in a distributed system

## Problem statement
What is the main advantage of using eventual consistency in a distributed system?

## Options: Pick one correct answer from below

    - Immediate data consistency across all nodes

    - Increased system availability

    - Simplified data management

    - Lower latency

---

# Solution description
*2* --> Eventual consistency allows the system to be more available since nodes can respond to requests without needing to wait for immediate consistency.
