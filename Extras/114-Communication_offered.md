# Communication offered

Problem statement
Communication offered by TCP is ________.

Options: Pick one correct answer from below

Full-duplex

Half-duplex

Semi-duplex

Byte by byte

Solution description
*1* --> Data can flow both the directions at the same time during a TCP communication hence, it is full-duplex. This is the reason why TCP is used in systems that require full-duplex operation such as e-mail systems.
