# Sync or Async

Problem statement
You want to develop a note-taking application like Google Docs where multiple people can write, read and update simultaneously without worrying about the access of the notes or passing the updated document in a group. Which type of communication would you use?

Options: Pick one correct answer from below

Synchronous Communication

Asynchronous Communication

Solution description
*1* --> For a note-taking application where multiple people can write, read, and update simultaneously, you should use synchronous communication. This approach ensures that all users see updates in real-time, allowing for instant collaboration and immediate reflection of changes across all instances of the document. Synchronous communication facilitates a live, interactive experience where users can work together without delays or the need to manually share updated versions.
