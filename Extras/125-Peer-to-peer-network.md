# Peer-to-peer network

Problem statement
Which of the following is a characteristic of a peer-to-peer (P2P) network architecture?

Options: Pick one correct answer from below

Centralized control

Single point of failure

Nodes act as both clients and servers

Simplified data management

Solution description
*3* --> In a P2P network, each node can act as both a client and a server, distributing tasks and resources across the network.
