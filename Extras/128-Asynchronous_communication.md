# Asynchronous communication

Problem statement
In the context of system design, which of the following is a benefit of using asynchronous communication between services in a distributed system?

Options: Pick one correct answer from below

Lower latency

Reduced system complexity

Improved service decoupling

Simplified service discovery

Solution description
*3* --> Asynchronous communication allows services to operate independently without waiting for responses, improving decoupling and system resilience.
