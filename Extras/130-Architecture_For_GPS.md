# Architecture For GPS

Problem statement
In the context of system design, an online gaming application uses inbuilt functionalities like GPS service of the device. Which architectural style is suitable for this application:

Options: Pick one correct answer from below

Pooled architecture

Converged architecture

Service Oriented Architecture

Integrated Architecture

# Solution Description
*3* --> SOA structures application components as independent, loosely coupled services. This makes it highly efficient to plug in external hardware sensors and software functionalities—such as GPS data—into the core game server.
