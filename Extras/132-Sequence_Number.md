# Sequence Number

Problem statement
Suppose a TCP connection is transferring a file of 1000 bytes. The first byte is numbered 10001. What is the sequence number of the segment if all data is sent in only one segment?

Options: Pick one correct answer from below

10000

10001

12001

11001

Solution description
*2* --> The sequence number given to first byte of a segment, with respect to its order among the previous segments, is the sequence number of that segment.
