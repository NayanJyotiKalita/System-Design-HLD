HTTP Request

Problem statement
The HTTP request message is sent in _________ part of three-way handshake.

Options: Pick one correct answer from below

First

Second

Third

Fourth

Solution description
*3* --> In first step client sends a segment to establish a connection with the server. In the second the step the client waits for the acknowledgement to be received from the server. After receiving the acknowledgement, the client sends actual data in the third step.
